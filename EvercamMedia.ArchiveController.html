    GET "/v1/cameras/:id/archives", ArchiveController, :index

    index method is being used to enlist all archives of a given camera.

    This endpoint is going through auth. So It will work by sending an HTTP request with API_KEY and API_ID.

    Example and Code Explanation

    GET /v1/cameras/dexter/archives?api_key={Your_API_key}&api_id={Your_API_id}

    The resulting object will be list which will contain all the archives of a given camera.
